Project is created with:
* Node.js (NPM)
* [kata.ai](https://kata.ai/)
* [Line](https://developers.line.biz/en/)

# Installation
open `Terminal(Linux,Mac)` or `CMD(Windows)`
```
$ npm install -g kata-cli
$ kata login
$ kata select-project
$ kata push
```
